# 🚀 ProctorX – AI-Powered Online Examination & Proctoring Platform


## 📖 Overview

**ProctorX** is an advanced AI-powered online examination and remote proctoring platform designed to conduct secure, scalable, and intelligent assessments. The platform combines examination management, real-time candidate monitoring, automated evaluation, integrity analysis, and detailed reporting into a single unified ecosystem.

Built with modern web technologies, ProctorX enables educational institutions, recruitment organizations, training centers, and enterprises to conduct examinations remotely while maintaining fairness, transparency, and academic integrity.

The system utilizes intelligent monitoring mechanisms such as webcam analysis, screen-sharing verification, browser monitoring, face detection, gaze tracking, and behavioral analysis to identify suspicious activities during examinations.

# 🏗️ System Architecture

ProctorX consists of two major modules:

## 👨‍💼 Admin Module

The administrator manages assessments, students, monitoring activities, reports, and overall platform operations.

## 👨‍🎓 Student Module

Students can securely participate in examinations, monitor performance, review results, and track integrity scores.

# 🔍 Secure Exam Verification Process

Before entering an examination, every student must complete an 8-step security verification workflow.

---

## Step 1 – Instructions

Students review examination guidelines, rules, and prohibited activities.

---

## Step 2 – Browser Verification

The platform verifies browser compatibility and security requirements.

---

## Step 3 – Webcam Verification

The webcam is tested and facial visibility is confirmed.

---

## Step 4 – Microphone Verification

Audio monitoring capability is validated.

---

## Step 5 – Screen Sharing Verification

Students must share their entire screen for continuous monitoring.

---

## Step 6 – Network Verification

Internet connectivity and network stability are checked.

---

## Step 7 – Identity Photo Capture

A live photograph is captured for identity verification.

---

## Step 8 – Verification Summary

A final review screen displays verification results.

Only after successful completion of all checks is the student allowed to begin the examination.

# 🚀 Quick Start

## Backend Setup

```bash
cd backend

cp .env.example .env

npm install

npm run seed

npm run dev
```

Backend server starts at:

```bash
http://localhost:5000
```

---

## Frontend Setup

```bash
cd frontend

cp .env.example .env

npm install

npm run dev
```

Frontend application starts at:

```bash
http://localhost:5173
```
